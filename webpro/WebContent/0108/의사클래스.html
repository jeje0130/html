의사클래스 : 상태나 구조에 의해서 선택이 이루어진다 
		- 상태 : :link, :visited, :hover, :active
		- p태그에는 :link, :visited 안씀
		
		a:link {
		color : red;
		text-decoration : none;
		}
		a:visited {
			color : green;
		}
		a:hover{
			color : blue;
			text-decoration : underline;
		}
		a:active{
			color : black;
			font-size : 1.2em;
		}

무궁화 꽃이 피었습니다

	구조에 의한 의사코드
	td:nth-child(3) : tr에서 3번째 자식을 선택
	td:nth-child(even) : 짝수번째 자식
	td:nth-child(odd) : 홀수번째 자식
	td:nth-child(2n) : 짝수번째 자식

	#tab1{
		border : 2px solid blue;
		border-collapse : collapse;
	}
	#tab1 td{
		width : 150px;
		height : 40px;
	}
	#tab1 .t1{
		background : skyblue;
	}

	#tab2{
		border : 5px inset purple;
		border-spacing : 10px;     /* spacing : 벌리는 것 */
	}
	#tab2 td{
		width : 150px;
		height : 30px;
	}
	#tab2 tr:nth-child(even){      /* 짝수번째 tr */
		background : pink;
	}

	#tab3{
		border : 4px double orange;
	}
	#tab3 td{
		width : 160px;
		height : 50px;
	}
	#tab3 td:nth-child(odd){      /* 홀수번째 td */
		background : lightgray;
	}
